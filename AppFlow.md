omplete App Flow
1. Account Creation Flow

Step 1. Sign Up
User enters
• Name
• Email
• Password
Then selects Next.

Step 2. Upload Photo
User uploads a profile image.

Step 3. Select Role
User selects a role such as designer, developer, manager, etc.

Step 4. Workspace Setup
User enters
• Workspace name (team name, for example Milestoro Team)
• Phone number

After this, the account is created and the profile is fully set up.
User is taken to the Dashboard.

2. Dashboard Features

The dashboard contains everything for the team.
One important rule: Chat is team wide, not per project. All team members share one chat space.

3. Folder and Project Structure
A. Create Folder

When creating a new folder
• Enter folder name
• Select folder color

A folder contains multiple projects.

B. Create Project

Inside any folder, the user can create a project by entering
• Project name
• Project date
• Project color

Projects live inside folders, and tasks live inside projects.

4. Task Creation

Inside a project, when creating a task the user must enter
• Task title
• Description
• Due date
• Priority level
• Assign to a team member (optional)
• Attach a file (optional)

After creating the task
• It must appear inside the project
• It must also appear in the calendar on the correct due date

Calendar sync is required. Tasks are always synced with their dates in the calendar.

5. Calendar Integration

Every task with a due date is automatically placed on that calendar day.
If a task is updated, the calendar updates as well.
